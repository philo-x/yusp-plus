# BASE-001 阶段时效与扣减

> 来源场景：`SCN-BASE-001`
>
> 状态：DRAFT / 待联合评审

## 用户目标

准确计算需求阶段停留，对有依据的暂停时间发起审批扣减，并能从时间区间、业务日历和审批记录重算。

## 规范性需求

| ID | 优先级 | 需求 |
|---|---|---|
| FR-BASE-001 | Must | J06 每日 00:00 计算在建需求阶段停留；查询可实时展示但不得重复累计。 |
| FR-BASE-002 | Must | 有效停留天数=阶段业务天数−已批准扣减天数，结果不得小于 0。 |
| FR-BASE-003 | Must | PM 提交阶段、天数和原因，只有部落长批准后扣减生效，审批全程留痕。 |
| FR-BASE-004 | Must | 回退后再次进入同一阶段时保留并累计各独立区间，不覆盖历史。 |

## 接口、数据与作业

- 接口：`IF-065`～`IF-070`。
- 数据：`DATA-REQ`、`DATA-BASE`。
- 作业：`J06`。

## 非范围

- 未审批扣减直接生效。
- 删除或覆盖阶段区间和审批历史。
- 在业务日历口径批准前自行决定周末、节假日和半天假算法。
