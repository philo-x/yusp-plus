# BASE-001 验收标准

## AC-BASE-001：时效计算、扣减和回退

- **Given** 正常、跨周末/节假日、回退重入、多次扣减、未批准和超额样本。
- **When** 首次/重复执行 J06、实时查询，并提交批准/拒绝扣减。
- **Then**
  - 同日重跑不重复累计；
  - 有效天数可由独立区间、批准日历和已批准扣减重算，且不小于 0；
  - 未批准、负数、超额和无权申请不生效；
  - 回退重入的每个区间与审批历史可追溯。

## 通过条件

- 覆盖 `FR-BASE-001～004`、`IF-065～070` 和 J06 幂等。
- `TBD-BASE-001` 未关闭前，业务日历相关边界不得最终通过。
- 证据须包含输入区间、日历版本、审批记录、计算明细和重跑对比。
