# MIG-001 验收标准

## AC-MIG-001：全流程迁移、续传与回滚

- **Given** 批准的脱敏类生产快照、映射、冻结点、增量水位、批次和回滚方案。
- **When** 完整迁移，期间人为中断、从检查点续传、同输入重跑并回滚指定批次。
- **Then** 范围、顺序、活动/归档存储正确；无重复或悬空；源数据不变；冲突转人工；所有动作可审计。

## 全量校验

| ID | 检查 | 通过条件 |
|---|---|---|
| MIG-VAL-001 | 记录数 | 源数=成功数+批准排除数+异常数，切换前异常数为 0。 |
| MIG-VAL-002 | 关键字段 | 业务标识、状态、日期、责任人、组织、分钟和基线状态逐条一致。 |
| MIG-VAL-003 | 关联完整性 | 需求—阶段/任务、人员—组织/报工、任务—报工无非预期悬空。 |
| MIG-VAL-004 | 业务汇总 | 按月、组织、人员、需求汇总的工时、需求数和基线数前后一致。 |
| MIG-VAL-005 | 抽样复核 | 至少 10% 附加人工或独立复核，不替代全量校验。 |
| MIG-VAL-006 | 可重复性 | 同输入重跑不新增重复对象，校验结果稳定。 |

## 通过门槛

- 六项校验、续传、重跑和回滚演练全部通过。
- 业务、数据、测试和运维共同签署。
- 无未接受的数据异常或版本冲突。
