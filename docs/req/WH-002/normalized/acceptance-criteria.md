# WH-002 验收标准

## AC-WH-004：管理权限与知脉写回

- **Given** 各管理角色数据范围真值集和成功、超时、重复、部分失败样本。
- **When** 查询、筛选、导出管理报工并执行/重试 J04。
- **Then** 各角色只看到批准范围；越权 ID 请求被拒绝；同步状态和重试可查；超时不伪成功；重复不重复核算；部分失败有逐项结果。

## AC-WH-002-ROUTE：灰度来源裁决

- **Given** 知效/知微同一人员日期的单任务、多任务、重复和来源冲突数据。
- **When** J05 按生效规则处理。
- **Then** 每个人员日期只有一个权威来源；该来源全部合法明细保留；另一来源不入账；裁决证据可对账。

- `TBD-JOB-001/002` 和 `TBD-ROL-002` 未关闭前，相关时序/冲突场景不得最终通过。
