# TASK-002 验收标准

## AC-TASK-003：同步、展示和确认

- **Given** 首次、重复、版本更新、无法映射的知安事件，以及责任/非责任开发人员。
- **When** 同步安全内容、查看标识、尝试编辑、执行单项/批量确认并模拟知安不可用。
- **Then**
  - 同一源任务幂等，版本历史保留，无法映射进入异常；
  - 安全内容与源一致且不能编辑；
  - 只有批准主体可确认，未确认状态不能完成；
  - 首次确认只产生一个业务结果，重复和重试不重复；
  - 外部失败保留可追踪待处理状态。

## 通过条件

- 覆盖 `FR-TASK-006/008/012/013`、`FR-CROSS-001/002`。
- 单项和批量确认、权限拒绝、重试耗尽和版本更新均有证据。
- `TBD-TASK-001` 未关闭前，安全任务计工场景不得最终通过。
