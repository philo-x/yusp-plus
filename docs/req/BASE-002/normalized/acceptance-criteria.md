# BASE-002 验收标准

## AC-BASE-002：基线、取消、加塞与调出

- **Given** 当月/非当月计划需求，覆盖 5 日、6 日、30 日、月末和跨年。
- **When** 按时刻首次及重复执行 J07/J08/J09，并在 5 日前后变更计划月。
- **Then** 各分类条件和边界正确；同需求同月结果唯一；重跑不重复；调出保留历史。

## AC-BASE-003：原计划与计划变化

- **Given** 原计划为空、当月已有、跨月已有及多次计划变更数据。
- **When** 首次/再次纳入、取消并查询首次上线、偏差和变化。
- **Then** 原计划只首次保存；取消只删当月值；跨月值保留；首次上线不覆盖；变化历史可还原。

## 通过条件

- 覆盖 `FR-BASE-005～011`、`IF-071～081` 和 J07～J09。
- 重复执行、并发和月度边界均有证据。
- 未明确的 30 日后规则不得通过实现假设加入验收。
