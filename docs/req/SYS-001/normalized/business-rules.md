## 业务规则列表 (Business Rules)

- **`BR-186`**：**监管结论不变原则**：对于反洗钱、EAST5.0 数据治理及数据脱敏，严格维持 PRD 原始结论“不涉及”，研发团队禁止擅自增加额外的文本脱敏掩码规则干扰用户报工或查看。
- **`BR-187`**：**审计日志完整性**：对于修改、删除及敏感数据导出操作，`modify_trace_log` 必须保存变更前后 `before_data` 与 `after_data` 的完整 JSON 镜像，审计日志保留时间不低于 3 年。
- **`BR-188`**：**安全控制阻断原则**：在 `Q-12` 应用安全控制规约正式完成评审补充前，系统技术架构不得擅自改变加密方式。
