# SYS-001 非功能、安全与合规要求 - 验收标准说明书 (acceptance-criteria.md)

> **任务包编号**：SYS-001  
> **任务包名称**：非功能、安全与合规要求  

---

## 验收标准列表 (Acceptance Criteria)

- **`AC-266`**：**基准性能压测验证**：使用 JMeter/Gatling 在 5,000 在线用户并发场景下压测需求列表及报工提交，TPS >= 500 时 95% 响应时间 <= 300ms，系统 0 报错。
- **`AC-267`**：**审计追溯验证**：进行一次需求批量修改操作，查询 `modify_trace_log` 表，新增一条包含 `operator_id`、`IP` 及变更前后差异 JSON 的记录。
- **`AC-268`**：**创新改造 GoldenDB 兼容性测试**：在 GoldenDB 6.x 环境下运行全量后端单元测试与集成测试，73 张表 SQL 语法无死锁、无报错。
