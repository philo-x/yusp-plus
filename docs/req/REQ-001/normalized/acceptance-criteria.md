# REQ-001 验收标准

## AC-REQ-001：三视图、筛选和个人显示

- **Given** 包含 13 阶段、缺计划、超期和归档需求的真值集及两个不同权限用户。
- **When** 用户在三种视图间切换、组合筛选、保存个人筛选并配置列。
- **Then** 三种视图集合一致；字段和状态与详情/源数据一致；个人配置持久且互不影响；缺计划和归档边界正确。

## AC-REQ-003-VIEW：横幅与组织视角

- **Given** 风险、超期、紧急需求及多组织权限真值集。
- **When** 读取横幅、钻取指标并尝试授权/越权组织切换。
- **Then** 聚合与明细一致；钻取正确；越权组织不返回且直接调用被拒绝。

## AC-REQ-004：自定义字段

- **Given** 四级作用域、六类字段和并发配额请求。
- **When** 创建、编辑、归档、赋值、筛选、导入和导出。
- **Then** 类型与作用域正确；第 11/21 个有效字段被拒绝；归档历史保留；导入导出列正确；外发数据不含自定义字段。

## 通过条件

- 覆盖本包全部 Must 需求与 `DATA-RULE-001/003/004`。
- 页面和直接接口授权结果一致。
- 证据包含环境、数据集版本、执行人、日期和追踪号。
