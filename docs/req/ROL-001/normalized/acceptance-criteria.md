# ROL-001 验收标准

## AC-ROL-001：白名单、扩围与回滚

- **Given** 三种白名单、同人多规则、双来源同日多任务、重复/冲突提交和故障门槛样本。
- **When** 变更白名单、运行 J05、申请扩围并触发冻结/回滚阈值。
- **Then**
  - 三种粒度可配置、版本化、可审计，未命中用户走知微；
  - 每个人员日期仅采用一个权威来源，合法明细不折叠；
  - 裁决保存候选摘要、来源、规则版本、明细数和分钟合计；
  - 无指标证据/批准不能扩围，达到阈值停止新流量；
  - 回滚后知微可承接，期间数据、补偿和冲突清单完整；
  - 未授权操作被拒绝并记录。

## AC-ROL-OBS：可观测性

- 注入成功、业务失败、超时、积压和回滚事件。
- 指标、结构化日志、追踪号和告警可定位批次、人员日期、规则版本及原因。
- 告警字段完整且不暴露敏感原文。
