> **任务包名称**：需求与任务 - 自定义字段管理  

---

## 业务规则列表 (Business Rules)

- **`BR-061`**：**自定义字段数量配额硬约束**：
  - 全局作用域自定义字段数量上限：`<= 20`。
  - 单个项目私有自定义字段数量上限：`<= 10`。
  - 达到上限后新增保存将直接抛出业务异常。
- **`BR-062`**：**字段停用非破坏原则**：字段停用采用归档标记（`status = ARCHIVED`），严禁逻辑或物理删除底层配置及 historical value 数据。
- **`BR-063`**：**本地化与度量隔离约束**：自定义字段仅作为知效平台内部辅助管理信息，不得写入对外 API 报文、不得发送至 MQ、不得包含在效能度量大屏统计 SQL 中。
- **`BR-064`**：**权限匹配规则**：创建和修改自定义字段的权限必须严格遵照 PRD 3.5 节矩阵控制，项目经理仅可操作本项目私有字段。
