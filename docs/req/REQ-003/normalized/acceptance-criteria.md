# REQ-003 需求管理 - 固定横幅与指标统计 - 验收标准说明书 (acceptance-criteria.md)

> **任务包编号**：REQ-003  
> **任务包名称**：需求管理 - 固定横幅与指标统计  

---

## 验收标准列表 (Acceptance Criteria)

- **`AC-056`**：**红色加粗样式验证**：超期/受阻需求数在 UI 上渲染为红色字体（HEX `#FF4D4F` 或 CSS `color: red`）且 CSS `font-weight >= 600`。
- **`AC-057`**：**横幅置顶吸顶测试**：在需求列表中滚动下翻 500 像素，顶部横幅保持 CSS `position: sticky` 吸顶展示，不随页面滚动移出视窗。
- **`AC-058`**：**在建紧急需求点击联动测试**：横幅显示在建紧急需求数为 5；点击该数字，需求列表条件区自动挂载 `is_urgent = true` 标签，列表仅保留该 5 条需求。
- **`AC-059`**：**高亮提示测试**：手动添加筛选条件“需求类型 = 一般需求”，横幅筛选指示器显示高亮背景框（CSS 高亮状态）。
- **`AC-060`**：**切换组织刷新测试**：管理层由“零售部落”切换为“企金部落”，横幅中的总需求数、超期数、紧急数瞬间刷新为“企金部落”的统计值。
